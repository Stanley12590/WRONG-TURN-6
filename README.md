# WRONG TURN 6 WhatsApp Bot

Multi-device WhatsApp bot developed by STANYTZ with pairing code system.

## Features
- ✅ Multi-user support (each user with their own number)
- ✅ Pairing code system (no QR scanning needed)
- ✅ Beautiful iOS-style web interface
- ✅ Admin dashboard for monitoring
- ✅ Auto-reconnect on disconnect
- ✅ MongoDB session storage
- ✅ Rate limiting for security
